Trail.

- April 2021

Trail is a Virtual Tour Guide Application that gives users the ability to learn about the trees located around the Heriot-Watt University Campus. This was a 4th year dissertation project completed towards my BSc Computer Science Degree. It acheived a grade of 84%.

Trail was developed as a Progressive Web Application (PWA) which allowed users to install the web app to their home screen and run it "natively" on their device. Users can undertake a tour via the app and learn about trees in a variety of innovative ways such as: Spatial Triggering (triggering tree information by walking passed it), Augmented Reality (immersive viewing of tree information via a custom AR experience), QR Code Scanning (In-built scanner within app allowing user to scan custom QR codes attached to trees), Speech Synthesis (have tree information automatically read aloud to the user), and UI Map interaction (custom interactive tree map).
